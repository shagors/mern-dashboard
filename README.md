## Client side design:
    * React
    * react-redux for data management and toolkit
    * Design for mui with icons, material, emotion-styled
    * for table design x-data-grid
    * for [chart design](https://nivo.rocks/)- bar, core, pie, line, geo,
    * react date picker

## Server side design
    * Nodejs
    * Express.js
    * Mongodb
    * Mongoose
    * cors
    * dotenv
    * helmet
